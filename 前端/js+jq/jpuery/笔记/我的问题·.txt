            链式编程操作机制  链式操作的本质是：每个方法返回一个新的jQuery对象，该对象表示当前操作上下文中的元素集合
                案例:$(this).stop().animate({width: 224}).find(".small").stop().fadeOut().siblings(".big").stop().fadeIn();
                分析:$(this)：
            这是一个jQuery对象，指向事件触发时的元素（假设为一个li元素）。例如，在您的引用[1]中，$(this)常用于事件处理程序2。
                .stop().animate({width: 224})：
            对$(this)元素应用动画效果（停止当前动画后，将其宽度变为224px）。此方法返回原始的$(this)对象（即同一个元素集合），因此上下文未改变。
                .find(".small")：
            这是关键转折点。.find()方法在$(this)元素内部查找类名为.small的子元素（例如，在您的引用[3]中，.small定义为绝对定位的子元素3）。.find()返回一个新的jQuery对象，该对象包含.small元素（如一个img或div）。此时，上下文转移到.small元素上。
                .stop().fadeOut()：
            对.small元素应用淡出动画。方法返回的仍是.small元素的jQuery对象，上下文保持不变。
                .siblings(".big")：
            这里，.siblings()方法作用于当前上下文元素——即.small元素。它查找.small元素的兄弟元素（同级元素），并筛选出类为.big的元素。例如，在您的引用[3]中，.big类定义为.small的兄弟元素（都位于a标签下）

切割字符串函数:
    1. split() 方法
    用于将字符串按照指定的分隔符切割成数组。
        let str = "apple,banana,orange";
        let result = str.split(","); // ["apple", "banana", "orange"]
    2. substring() 方法
    用于提取字符串中介于两个指定下标之间的部分。
        let str = "Hello, World!";
        let result = str.substring(0, 5); // "Hello"
    3. substr() 方法
    用于从指定位置开始，提取指定长度的子字符串。
        let result = str.substr(7, 5); // "World"
        4. slice() 方法
    类似于 substring()，但支持负索引。
        let result = str.slice(-6, -1); // "World"
